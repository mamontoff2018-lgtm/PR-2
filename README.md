### Ответы на контрольные вопросы

1. **Что такое XML? Для каких целей он используется в Android-разработке?**  
   XML (eXtensible Markup Language) — язык разметки, используемый в Android для декларативного описания пользовательского интерфейса. Позволяет описывать структуру экрана в отдельных файлах вместо программного создания элементов в коде.

2. **Что такое тег (элемент) в XML? Из каких частей он состоит?**  
   Тег — основной строительный блок XML. Состоит из открывающего тега `<Имя>`, закрывающего тега `</Имя>` и содержимого (атрибутов и вложенных элементов).

3. **Какие менеджеры размещения (контейнеры) вы знаете? Кратко опишите каждый.**  
   - `LinearLayout` — линейное расположение элементов (горизонтально или вертикально).  
   - `GridLayout` — табличное расположение в виде сетки строк и столбцов.  
   - `ConstraintLayout` — гибкое позиционирование относительно других элементов.  
   - `FrameLayout` — элементы накладываются друг на друга.  
   - `RelativeLayout` — позиционирование относительно родителя или других View.

4. **В чём разница между LinearLayout и GridLayout?**  
   `LinearLayout` располагает элементы в одну линию (ряд или столбец). `GridLayout` позволяет создавать двумерную сетку с возможностью объединения ячеек.

5. **Что такое match_parent и wrap_content?**  
   - `match_parent` — элемент занимает всё доступное пространство родителя.  
   - `wrap_content` — элемент занимает ровно столько места, сколько нужно для его содержимого.

6. **В чём разница между android:gravity и android:layout_gravity?**  
   - `android:gravity` — выравнивает содержимое **внутри** самого элемента.  
   - `android:layout_gravity` — выравнивает сам элемент **внутри** родительского контейнера.

7. **Какие единицы измерения используются в Android? Для чего предназначены dp и sp?**  
   - `dp` (density-independent pixels) — для размеров элементов (рекомендуется).  
   - `sp` (scale-independent pixels) — для размеров текста (учитывает масштаб шрифта пользователя).  
   - `px` — абсолютные пиксели (используется редко).

8. **Как создать простую фигуру (прямоугольник, круг) с помощью XML-ресурса в папке drawable?**  
   Создаётся файл с расширением `.xml` в папке `res/drawable`. Используется тег `<shape>` с атрибутами `android:shape="rectangle"` или `"oval"`, а также `<solid>`, `<corners>`, `<stroke>` и `<size>`.
